# Conversor de Moedas



<img src="https://github.com/lucassantoos96/conversor-de-moedas/blob/main/assets/img-conversor-readme.png?raw=true"/>



## Descrição
Projeto de conversor de moedas desenvolvido com HTML, CSS e JavaScript.  
Utiliza uma API para obter taxas de câmbio atualizadas em tempo real, permitindo converter valores entre diferentes moedas com precisão.

## Funcionalidades

- Conversão entre moedas como Real (BRL), Dólar (USD) e Euro (EUR)

- Consumo de API para obter taxas de câmbio atualizadas automaticamente

- Interface responsiva e intuitiva com HTML e CSS

## Tecnologias utilizadas

- HTML5

- CSS3

- JavaScript 

- API externa para taxas de câmbio
